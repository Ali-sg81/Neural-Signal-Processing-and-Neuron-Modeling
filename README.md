# Neural Signal Processing and Neuron Modeling

This project implements an end-to-end computational neuroscience pipeline for analyzing extracellular neural recordings. The workflow includes signal preprocessing, spike detection, waveform extraction, feature engineering, dimensionality reduction, clustering, and neuron simulation using the Brian2 simulator.

## Features

- Exploratory Data Analysis (EDA)
- Signal preprocessing
  - DC offset removal
  - High-pass filtering
  - Baseline correction
  - Noise analysis
- Threshold-based spike detection
- Spike waveform extraction
- Feature extraction
- PCA visualization
- K-Means / Gaussian Mixture clustering
- Leaky Integrate-and-Fire (LIF) neuron simulation using Brian2
- Comparison between simulated and biological neural activity
